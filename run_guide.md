# How to Run the ChatBot

## Method 1: Google Colab
1. Open [Google Colab](https://colab.research.google.com)
2. Install ChatterBot: `!pip install chatterbot==1.0.4`
3. Copy the code from `chatbot.py`
4. Run the code

## Method 2: Local Computer
1. Install Python 3.7+
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python chatbot.py`

## Troubleshooting
- If you get import errors, try: `pip install chatterbot==1.0.4 chatterbot-corpus`
- For newer Python versions, you might need to downgrade to Python 3.8
